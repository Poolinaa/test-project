# Amaya

Тестовое задание на Vue 3 + TypeScript + SCSS.

## Функционал

* Переключение между тарифами
* Хранение выбранной подписки в переменной
* Отображение выбранного тарифа в `alert` после нажатия на кнопку «Оформить подписку»
* Адаптация под мобильные устройства, планшеты и десктоп

## Стек

* Vue 3 (Composition API)
* TypeScript
* SCSS
* Vite

## Запуск проекта

Установка зависимостей:

```bash
npm install
```

Запуск dev-сервера:

```bash
npm run dev
```

## Структура проекта

```text
src/
├─ components/
│  ├─ Logo.vue
│  ├─ BenefitsCard.vue
│  ├─ PaywallButton.vue
│  └─ SubscriptionCard.vue
│
├─ data/
│  └─ plans.ts
│
├─ styles/
│  ├─ global.scss
│  └─ variables.scss
│
├─ App.vue
└─ main.ts
```

## Особенности реализации

* Данные тарифов вынесены в отдельный файл (`plans.ts`)
* Выбранный тариф хранится по `id`
* Компоненты разделены по зонам ответственности
* Для интерактивных элементов добавлены состояния
